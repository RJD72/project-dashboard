Reduced Width Page

Similar to our full page layout, the page is comprised of global header, nav, and footer elemets, along with the main content composed of cards. Depending on the viewport's size, the types of cards displayed will be limited, as well as the dropdown inputs. Using a combination of media queries and pseudo-classes, we have established a consistent and cohesive design across viewport sizes.

Content in a reduced width will be styled appropriately based on the viewport size
