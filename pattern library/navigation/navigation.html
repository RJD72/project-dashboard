Navigation

Header Navigation Example 1

The navigation is an important component to any application. A navigation is used to enable the user’s to navigate through a website and it is important that the navigation is coherent and functional. Adding classes to the website's navigation allows the developer to add styles like position, colours and responsiveness to the navigation. The class navbarExampleOne is there to add styling to this specific element. The class nav-links-example-one is to control the styling of the navigation links.

Header Navigation Example 2

For larger more complex websites there is a search function that is added to the navigation. The search function consists of a div with the class search-container and an input element with the class search-input and search-input:focus. The search-container element in this example is used for the positioning of the input element. The input element with the class search-input is used for the actual styling of the element like colours and borders. The element class also has search-input:focus which is used to style the input element when it is focused.

Sub Navigation

Header Sub Navigation Example 1

For a more complex website, we added a sub navigation bar to the top of the main navigation. The div with the class green-bar represents the sub navigation section that goes above the main navigation and contains the language dropdown, the Help/FAQ link, a phone number link and a Login/Register link.

Header Sub Navigation Example 2

For this sub navigation we’ve added a search input, this for when the website will become more complex when the user registers for an account. We’ve organized the sub navigation into left and right links. A div with the class name left-links and a div with the class name right-links.

Mobile Menu

Mobile Navigation Examples

This navigation is mobile responsive meaning that it is used when the website gets viewed on smaller screens like iPhones or tablets and it is functional and visually coherent. It has a div with the class mobile-menu-icon which holds three bars that represent the mobile menu toggle button. The onclick attribute is used to trigger a JavaScript function called toggleMobileMenu().
