Heading

XL Heading Example

The extra-large heading uses the h1 tags. It is the biggest font size used in our application. A heading of this size is usually used to grab the user's attention. The xl-heading-example class can be used to make unique customizations to just the extra-large heading. The headingFont class is used to assign the heading a font that is also used on all of the other headings.

Lg Heading Example

The large heading size is marked with h2 tags, it is the second largest font size used in our application. This size of heading is usually used for the titles of articles or paragraphs of content. The lg-heading-example class is there for when a user might want to customize all the large headings and the headingFont class is there to add the font style that is used on all headings throughout the application.

Md Heading Example

This medium sized heading is marked with h2 heading tags, the third largest font size used in our application. Medium font sizes like this can be used for labels, subtitles or other similar texts. The md-heading-example class is there for when a user might want to customize all the medium headings within the application and the headingFont class is there to add the font style that is used on all headings throughout the application.
