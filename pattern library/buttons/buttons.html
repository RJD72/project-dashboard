Button Elements

Buttons and dropdowns are essential user interface elements used for interaction and navigation. This documentation provides details on various button styles and their usage.

Usage

These button components can be used in various parts of your website for actions, links, or as part of dropdown menus. The different styles and color schemes allow for flexibility in design and interaction. Be sure to apply the appropriate classes and styles to achieve the desired appearance and functionality.

Custom Colored Buttons

  • These buttons are designed with various color schemes, each represented by a different class.
  • Example color schemes: GreenStarWing, RedPastel, OrangePastel, YellowPastel, GreenPastel, BluePastel, PurplePastel.
  • These buttons can be used for different actions or purposes based on their color scheme. 
        
  <button class="starwing-btn colourGreenStarWing">GreenStarWing</button>
  <button class="starwing-btn colourRedPastel">RedPastel</button>
  <button class="starwing-btn colourOrangePastel">OrangePastel</button>
  <button class="starwing-btn colourYellowPastel">YellowPastel</button>
  <button class="starwing-btn colourGreenPastel">GreenPastel</button>
  <button class="starwing-btn colourBluePastel">BluePastel</button>
  <button class="starwing-btn colourPurplePastel">PurplePastel</button>

Disabled State

  • Make buttons inactive by adding the disabled boolean attribute to any <button> element.
  • They are primarily used to indicate that a specific action is not available or cannot be performed. 
        
  <button class="starwing-btn" disabled>GreenStarWing</button>
  <button class="starwing-btn" disabled>RedPastel</button>
  <button class="starwing-btn" disabled>OrangePastel</button>
  <button class="starwing-btn" disabled>YellowPastel</button>
  <button class="starwing-btn" disabled>GreenPastel</button>
  <button class="starwing-btn" disabled>BluePastel</button>
  <button class="starwing-btn" disabled>PurplePastel</button>

Custom Buttons

  • The .starwing-btn classes are designed to be used with the <button> element. However, you can also use these classes on <a> or <input> elements (though some browsers may apply a slightly different rendering)
Go Somewhere 
        
  <a class="starwing-btn colourGreenStarWing" href="#">Go Somewhere</a>
  <input class="starwing-btn colourGreenStarWing" type="button" value="Input"/>

Dropdown Button

  • Any single .starwing-btn can be turned into a dropdown toggle with some markup changes. Here’s how you can put them to work with <button> elements
  • The dropdown button is styled with the .colourGreenStarWing class.
  • It contains a <span> element with a down arrow character (▾) to indicate its dropdown nature.
  • The dropdown items are listed within a <div> element with the class .dropdown-content
Item 1 Item 2 Item 3 
          
  <div class="starwing-dropdown">        
    <button class="starwing-dropbtn colourGreenStarWing">Dropdown <span>▾</span></button>
    <div class="dropdown-content">
      <a href="#">Item 1 </a>
      <a href="#">Item 2 </a>
      <a href="#">Item 3 </a>
    </div>
  </div>
